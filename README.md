# ShostkaAir

<b>Проект Shostka - разработан по заказу Шосткинского городского совета.</b><BR>
Проект бесплатный. Использую как площадку для тестирования фреймворков, технологий и подходов к разработке.<BR>
Первый commit 27.02.2019. За это время функционал и назначение несколько раз менялось, осталось много неиспользуемого и закомментированного кода. Например, функционал опроса и отображения графиков со станций мониторинга загрязнения воздуха (станции моей разработки).  <BR>
