# ShostkaAir


Проект Shostka - разработан по заказу Шосткинского городского совета.
Проект бесплатный. Использую как площадку для тестирования фреймворков, технологий и подходов к разработке.
Первый commit 27.02.2019. За это время функционал и назначение несколько раз менялось, осталось много не используемого и закомментированного кода. Например, функционал опроса и отображения графиков со станций мониторинга загрязнения воздуха (станции моей разработки).
